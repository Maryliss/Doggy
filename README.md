# Pré-requis
$ télécharger le zip
$ dézipper
$ chemin du dossier dans le terminal
$ bundle install (pour avoir les gems: gem 'table_print' et gem 'faker')
$ rails db:migrate
$ rails db:seed
$ rails c
$ 'tp dogsitter.all' /  'tp dog.all' / 'tp city.all' / 'tp stroll.all'
